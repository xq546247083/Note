# 分部视图、组件视图

>## 分部视图

---

    1、后台获取分部视图，代码如下：
        new PartialViewResult
        {
            ViewName = "_AuthorPartialRP",
            ViewData = ViewData,
        };
    2、在标记文件中使用分部视图
        1、<partial name="_PartialName.cshtml" />
        2、@await Html.PartialAsync("_PartialName", model, customViewData)
